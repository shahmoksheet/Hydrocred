## 🎨 **Design Guidelines:**

### **Colors:**
- **Primary Green:** #10B981 (Emerald)
- **Primary Blue:** #3B82F6 (Blue)
- **Dark Background:** #1F2937 (Gray-800)
- **Light Text:** #F9FAFB (Gray-50)
- **Accent:** #06B6D4 (Cyan)

### **Fonts:**
- **Headings:** Inter Bold, 32-48px
- **Body:** Inter Regular, 18-24px
- **Captions:** Inter Medium, 14-16px

### **Layout:**
- **Margins:** 40px all sides
- **Spacing:** 24px between elements
- **Alignment:** Center for titles, left for content
- **Icons:** Use consistent icon style (outline or filled)

### **Visual Elements:**
- **Charts:** Use bar charts, pie charts for data
- **Screenshots:** Include actual platform screenshots
- **Diagrams:** Use flowcharts for processes
- **Branding:** Consistent Hydro-Cred logo placement

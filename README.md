# HydroCred

A blockchain-based platform for managing and trading hydropower credits, promoting sustainable energy production and transparent certification.

  * We are addresses rising carbon emissions by creating a transparent green hydrogen credit trading system.
  * The solution is a blockchain-based system that tracks hydrogen production and consumption in real-time.
  * It utilizes React, Node.js, Express, and Ethereum/Polygon for its technology stack.
  * It aims to enable global green hydrogen trading and encourage sustainable industrial practices.
  * The team consists of visionary developers with expertise in AI, blockchain, and clean energy.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   ```bash
   npm install
   ```
2. Set up environment variables:
   - Create a `.env.local` file
   - Add your Gemini API key: `GEMINI_API_KEY=your_key_here`

3. Run the development server:
   `npm run dev`
